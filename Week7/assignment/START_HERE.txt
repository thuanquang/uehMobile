ğŸ› TROUBLESHOOTING
================================================================================

Problem: App won't build
Solution:
  - ./gradlew clean build
  - Check Java version: java -version (should be 11+)
  - Android Studio â†’ File â†’ Sync Now

Problem: RecyclerView is empty
Solution:
  - App is working! Empty state means no schedules yet
  - Click "Add Schedule" to create one

Problem: Notifications don't show
Solution:
  - Check app permissions in device settings
  - Ensure device time is correct
  - Schedule times should be in future
  - Check notification settings for app

Problem: Database errors
Solution:
  - Clear app data: Settings â†’ Apps â†’ Schedule Manager â†’ Storage â†’ Clear
  - Or: adb shell pm clear com.example.week7assignment
  - App will auto-create fresh database

Problem: Time Picker doesn't open
Solution:
  - Ensure device time zones are enabled
  - Try restarting the app
  - Check device locale settings
